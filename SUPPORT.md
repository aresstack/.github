# Support

## Getting Help

For usage questions and troubleshooting:

- Review the repository README and docs.
- Search existing issues and discussions.
- Open a new issue with reproducible details when needed.

## What to Include

To help maintainers respond quickly, include:

- Library name and version
- Java version and runtime details
- Build tool and relevant dependency snippets
- Steps to reproduce
- Expected vs. actual behavior

## Scope of Support

aresstack is maintained as an open-source effort. Response times may vary, and support is prioritized based on impact, clarity, and maintainability.
